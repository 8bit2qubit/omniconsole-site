Some games only speak keyboard and mouse. Nekomata weaves a charm for these apps, mapping XInput inputs to keyboard and mouse actions; each charm can also prevent double input by blocking the app's own native gamepad signals. Open the editor from the OmniCharm widget's "Customize gamepad mapping for this app…" button.
