Añade tus propias plataformas mediante URI de protocolo, ruta de ejecutable o aplicación empaquetada (MSIX / APPX), con una imagen de portada opcional. Los argumentos de lanzamiento están disponibles cuando se utiliza el tipo de ruta de ejecutable.
