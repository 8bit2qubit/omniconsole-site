OmniConsole 完整支持 **Xbox 兼容（XInput）控制器**操作。

## 设置界面

| 输入 | 动作 |
|---|---|
| **方向键 / 左摇杆** | 四向焦点导览 |
| **A 键** | 选取 |
| **B 键** | 退出 |
| **LB / RB** | 切换分类选项卡 |
| **Y 键** | 新增自定义平台（于自定义平台选项卡中） |
| **X 键** | 编辑聚焦的自定义平台（于自定义平台选项卡中） |
| **☰（Menu）** | 将聚焦平台设为默认并立即启动（FSE 模式中） |

## 手柄鼠标模式

把手柄变成鼠标与键盘。提供两种对应方式：**OmniNav** 与 **Classic**。光标速度可于设置中调整。

| 按键 | OmniNav | Classic |
|---|---|---|
| 左摇杆 | 光标 | 滚轮 |
| 右摇杆 | 滚轮 | 光标 |
| A | 鼠标左键 | Enter |
| B | 鼠标右键 | Esc |
| X | Page Down | Page Down |
| Y | Page Up | Page Up |
| LB | Ctrl+Shift+Tab | Tab |
| RB | Ctrl+Tab | 鼠标左键 |
| LT | Esc | Shift+Tab |
| RT | Enter | 鼠标右键 |
| LS | Shift+Tab | — |
| RS | Tab | — |
| 方向键 | 方向键（长按连发） | 方向键（长按连发） |

### 三种状态

- **关闭** — 禁用。
- **自动** — 于 Edge、Chrome、Firefox、Opera、Brave、Epic Games Store、文件资源管理器、Steam 启用。
- **强制开启** — 于所有应用启用。

可于 OmniCharm Game Bar 小组件切换。ROG Ally 家族自动禁用手柄鼠标模式 — 设备具备内置手柄映射。

## 手柄 Steam 快捷键

| 输入 | 动作 |
|---|---|
| **⧉（短按）** | 打开 Steam 菜单¹ |
| **⧉（长按）** | 打开快速访问菜单¹ |
| **☰（长按）** | 打开 Steam 游戏叠加界面²˒³ |

¹ 于 Steam Big Picture 中　² 游戏中　³ 可于 OmniCharm Game Bar 小组件切换

## OmniCharm 小组件

OmniCharm Game Bar 小组件可在游戏中切换手柄鼠标模式、变更控制器布局、调整光标速度、切换长按 ☰ 打开 Steam 游戏叠加界面。
