> ⚠️ **Change the FSE Home App setting _before_ uninstalling OmniConsole.** Windows does not automatically reset the FSE Home App setting when the app is uninstalled, which can cause **Task View to stop working** on some builds.

## Steps

1. Open **Settings → Gaming → Full Screen Experience**.
2. Set **Choose home app** to **Xbox** or **None**.
3. Right-click **OmniConsole** in the Start Menu and select **Uninstall**, or go to **Settings → Apps → Installed apps**.
4. Go to **Settings → Apps → Installed apps** and uninstall **OmniConsole OmniCharm** (the widget does not appear in the Start Menu).
