OmniConsole 完整支援 **Xbox 相容（XInput）控制器**操作。

在手把上按 **Xbox 鍵**可開啟 **Game Bar** 介面，長按則開啟**工作檢視**。

## 平台設定介面

| 輸入 | 系統平台 | 自訂平台 |
|---|---|---|
| **十字鍵 / 左搖桿** | 四向焦點導覽 | 四向焦點導覽 |
| **A 鍵** | 選取 | 選取 |
| **B 鍵** | 退出 | 退出 |
| **LB / RB** | 切換分類索引標籤 | 切換分類索引標籤 |
| **Y 鍵** | — | 新增自訂平台 |
| **X 鍵** | — | 編輯聚焦的平台 |
| **☰（Menu）** | 設為預設並立即啟動¹ | 設為預設並立即啟動¹ |

¹ 於 Xbox 模式 (FSE) 中

## 貓又模式

把手把變成滑鼠與鍵盤，還能隨每個程式量身自訂。提供兩種對應方式：**OmniNav** 與**經典**。游標速度可於設定中調整。

| 按鍵 | OmniNav | 經典 |
|---|---|---|
| 左搖桿 | 游標 | 滾輪 |
| 右搖桿 | 滾輪 | 游標 |
| A | 滑鼠左鍵 | Enter |
| B | 滑鼠右鍵 | Esc |
| X | Page Down | Page Down |
| Y | Page Up | Page Up |
| LB | Ctrl+Shift+Tab | Tab |
| RB | Ctrl+Tab | 滑鼠左鍵 |
| LT | Esc | Shift+Tab |
| RT | Enter | 滑鼠右鍵 |
| LS | Shift+Tab | — |
| RS | Tab | — |
| 十字鍵 | 方向鍵 | 方向鍵 |

### 開啟 / 關閉

**開啟 / 關閉**切換。開啟時你自訂的設定檔優先接手，下列程式則套用內建映射（同樣可自訂）：Edge、Chrome、Firefox、Opera、Brave、Epic Games Store、檔案總管、Steam、Discord。

可於 OmniCharm Game Bar 小工具切換。ROG Ally 家族自動停用貓又模式，裝置具備內建手把映射。

### 各程式映射

有些遊戲只認鍵盤與滑鼠，沒有原生手把操作。貓又為這類程式編一道咒，將每個 XInput 輸入位（A/B/X/Y、LB/RB、LT/RT、LS/RS、十字鍵、左搖桿、右搖桿）重新映射為鍵盤、滑鼠或游標動作，讓它們也能純手把遊玩。每道咒還能以貓又衝刺提高映射服務優先順序、為任一按鍵開啟長按連發，並阻擋程式自身的原生手把訊號，避免雙重輸入。

於 OmniCharm 小工具點選 **「自訂此程式的手把映射…」** 進入編輯器。

### 編輯器按鍵

| 輸入 | 設定檔清單 | 設定檔編輯器 |
|---|---|---|
| **十字鍵 / 左搖桿** | 焦點導覽 | 焦點導覽 |
| **A 鍵** | 編輯聚焦的設定檔 | 選取 |
| **B 鍵** | 退出 | 儲存並關閉 |
| **X 鍵** | 刪除聚焦的設定檔 | 刪除此設定檔 |
| **Y 鍵** | 在搜尋方塊與清單間切換焦點 | — |
| **LB / RB** | 上／下翻頁 | — |

### 可映射的目標

| 類別 | 目標 |
|---|---|
| 字母 | A–Z |
| 數字 | 0–9 |
| 功能鍵 | F1–F24 |
| 編輯／導覽／方向鍵 | Insert、Delete、Home、End、Page Up、Page Down、方向鍵、WASD |
| 控制鍵 | Enter、Esc、Tab、Backspace、Space、PrtSc、Pause、Caps Lock、Apps |
| 符號鍵 | ` - = [ ] \ ; ' , . / |
| 數字鍵盤 | Numpad 0–9，以及 . + - * / |
| 媒體鍵 | 音量、播放控制、瀏覽器鍵、啟動鍵 |
| 修飾鍵組合 | Ctrl / Shift / Alt / Win + 按鍵 |
| 滑鼠 | 左／右／中鍵、滾輪、游標移動、捲動 |

## 手把 Steam 快捷鍵

| 輸入 | 動作 |
|---|---|
| **⧉（短按）** | 開啟 Steam 選單¹ |
| **⧉（長按）** | 開啟快速存取選單¹ |
| **☰（長按）** | 開啟 Steam 遊戲內嵌介面²˒³ |

¹ 於 Steam Big Picture 中　² 遊戲中　³ 可於 OmniCharm Game Bar 小工具切換

## OmniCharm 小工具

OmniCharm Game Bar 小工具提供遊戲中的快速存取：

- **工作檢視**、**Xbox 媒體櫃**與 **Steam 內嵌介面**的一鍵捷徑。
- 切換**長按 ☰** 開啟 Steam 遊戲內嵌介面。
- 切換**貓又模式**、變更控制器版面（**OmniNav / 經典**）、調整游標速度。
- 透過 **「自訂此程式的手把映射…」** 按鈕，為前景程式開啟**貓又**各程式手把映射編輯器。
