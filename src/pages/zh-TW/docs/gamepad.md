OmniConsole 完整支援 **Xbox 相容（XInput）控制器**操作。

## 設定介面

| 輸入 | 動作 |
|---|---|
| **方向鍵 / 左搖桿** | 四向焦點導覽 |
| **A 鍵** | 選取 |
| **B 鍵** | 退出 |
| **LB / RB** | 切換分類索引標籤 |
| **Y 鍵** | 新增自訂平台（於自訂平台索引標籤中） |
| **X 鍵** | 編輯聚焦的自訂平台（於自訂平台索引標籤中） |
| **☰（Menu）** | 將聚焦平台設為預設並立即啟動（FSE 模式中） |

## 手把滑鼠模式

把手把變成滑鼠與鍵盤。提供兩種對應方式：**OmniNav** 與 **Classic**。游標速度可於設定中調整。

| 按鍵 | OmniNav | Classic |
|---|---|---|
| 左搖桿 | 游標 | 滾輪 |
| 右搖桿 | 滾輪 | 游標 |
| A | 滑鼠左鍵 | Enter |
| B | 滑鼠右鍵 | Esc |
| X | Page Down | Page Down |
| Y | Page Up | Page Up |
| LB | Ctrl+Shift+Tab | Tab |
| RB | Ctrl+Tab | 滑鼠左鍵 |
| LT | Esc | Shift+Tab |
| RT | Enter | 滑鼠右鍵 |
| LS | Shift+Tab | — |
| RS | Tab | — |
| 方向鍵 | 方向鍵（長按連發） | 方向鍵（長按連發） |

### 三種狀態

- **關閉** — 停用。
- **自動** — 於 Edge、Chrome、Firefox、Opera、Brave、Epic Games Store、檔案總管、Steam 啟用。
- **強制開啟** — 於所有應用程式啟用。

可於 OmniCharm Game Bar 小工具切換。ROG Ally 家族自動停用手把滑鼠模式 — 裝置具備內建手把映射。

## 手把 Steam 快捷鍵

| 輸入 | 動作 |
|---|---|
| **⧉（短按）** | 開啟 Steam 選單¹ |
| **⧉（長按）** | 開啟快速存取選單¹ |
| **☰（長按）** | 開啟 Steam 遊戲內嵌介面²˒³ |

¹ 於 Steam Big Picture 中　² 遊戲中　³ 可於 OmniCharm Game Bar 小工具切換

## OmniCharm 小工具

OmniCharm Game Bar 小工具可在遊戲中切換手把滑鼠模式、變更控制器版面、調整游標速度、切換長按 ☰ 開啟 Steam 遊戲內嵌介面。
