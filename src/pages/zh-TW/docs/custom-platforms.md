OmniConsole 內建一組系統平台，你也可以新增自己的：任何執行檔或應用程式皆可。一切操作都透過圖形介面完成，不需要手動編寫任何 JSON 設定檔。輸入方塊聚焦時，Windows 11 遊戲控制器鍵盤會自動彈出，無需實體鍵盤也能填寫所有欄位。

> ⚠️ **自訂平台為實驗性功能。** 內建平台經過測試，但你自行配置的自訂平台則未經測試。配置錯誤的平台可能無法正常啟動，某些情況下還會導致 Xbox 模式 (FSE) 環境不穩定，要恢復可能需要重設應用程式。首次開啟此功能時，OmniConsole 會顯示一份須先接受的同意聲明。

## 新增平台

在設定畫面開啟**自訂**索引標籤。按下手把的 **Y 鍵**，或點選螢幕下方的 **Y 新增** 提示列，即可開啟新增平台的對話方塊。

## 啟動類型

自訂平台採用以下三種啟動類型之一：

| 類型 | 說明 |
|---|---|
| 執行檔 | 指向一個 `.exe` 檔案。支援自訂啟動參數。內建手把相容的檔案選擇器，無滑鼠也能瀏覽。 |
| Protocol URI | 透過協定連結啟動，例如 `steam://`。 |
| 封裝應用程式 | 適用於 MSIX / APPX 應用程式（UWP 或 Microsoft Store 應用程式）。以套件家族名稱搜尋已安裝的套件。 |

## 卡片封面

你可以為自訂平台加上專屬的卡片封面圖。建議解析度為 800 × 560 px（10:7 比例）；圖片會自動縮放至該尺寸。

## 匯入與匯出

要分享平台，於卡片上點右鍵（或長按），選擇**匯出**，配置便會以 JSON 格式複製到剪貼簿。要加入他人分享的平台，使用自訂索引標籤的**匯入**按鈕並貼上 JSON。
