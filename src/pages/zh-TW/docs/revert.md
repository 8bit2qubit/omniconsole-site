> ⚠️ **解除安裝前，請務必先變更 FSE 首頁應用程式設定。** 解除安裝應用程式時，Windows 不會自動重置 FSE 首頁應用程式設定，可能導致部分版本的**工作檢視無法正常開啟**。

## 步驟

1. 前往 **設定 → 遊戲 → 全螢幕體驗**。
2. 將「選擇首頁應用程式」改為 **Xbox** 或 **無**。
3. 在開始功能表中對 **OmniConsole** 按右鍵選擇 **解除安裝**，或前往 **設定 → 應用程式 → 已安裝的應用程式** 解除安裝。
4. 前往 **設定 → 應用程式 → 已安裝的應用程式** 解除安裝 **OmniConsole OmniCharm**（小工具不會出現在開始功能表）。
