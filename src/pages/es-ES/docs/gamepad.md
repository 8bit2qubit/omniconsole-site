OmniConsole es totalmente operativo con un **mando compatible con Xbox (XInput)**.

En tu mando, pulsa el **botón Xbox** para abrir la interfaz superpuesta de la **Game Bar**, o mantenlo pulsado para abrir la **Vista de tareas**.

## Interfaz de Ajustes

| Entrada | Acción |
|---|---|
| **Cruceta / Stick izquierdo** | Navegar por el foco en las cuatro direcciones |
| **A** | Seleccionar |
| **B** | Salir |
| **LB / RB** | Cambiar entre las pestañas de categorías |
| **Y** | Añadir plataforma personalizada (en la pestaña Plataformas personalizadas) |
| **X** | Editar la plataforma personalizada seleccionada (en la pestaña Plataformas personalizadas) |
| **☰ (Menú)** | Establecer la plataforma seleccionada como predeterminada e iniciarla (en el Modo Xbox (FSE)) |

## Modo de ratón por mando

Convierte tu mando en un ratón y teclado. Hay dos distribuciones disponibles: **OmniNav** y **Clásica**. La velocidad del cursor se puede regular en los Ajustes.

| Botón | OmniNav | Clásica |
|---|---|---|
| Stick izquierdo | Cursor | Desplazamiento (Scroll) |
| Stick derecho | Desplazamiento (Scroll) | Cursor |
| A | Clic izquierdo | Enter |
| B | Clic derecho | Esc |
| X | Avanzar página | Avanzar página |
| Y | Retroceder página | Retroceder página |
| LB | Ctrl+Shift+Tab | Tab |
| RB | Ctrl+Tab | Clic izquierdo |
| LT | Esc | Shift+Tab |
| RT | Enter | Clic derecho |
| LS | Shift+Tab | — |
| RS | Tab | — |
| Cruceta (D-pad) | Teclas de dirección | Teclas de dirección |

### Tres estados

- **Desactivado** — inhabilitado.
- **Automático** — se activa en Edge, Chrome, Firefox, Opera, Brave, Epic Games Store, Explorador de archivos, Steam y Discord.
- **Forzar encendido** — se activa en todas las aplicaciones.

Se puede alternar a través del widget de Game Bar OmniCharm. El Modo de ratón por mando se desactiva automáticamente en los dispositivos de la familia ROG Ally, ya que cuentan con su propio mapeo de mandos integrado.

## Accesos directos de Steam para mando

| Entrada | Acción |
|---|---|
| **⧉ (pulsación corta)** | Abrir el menú de Steam¹ |
| **⧉ (pulsación larga)** | Abrir el menú de acceso rápido¹ |
| **☰ (pulsación larga)** | Abrir la interfaz superpuesta en el juego de Steam²˒³ |

¹ En Steam Big Picture ² En el juego ³ Configurable a través del widget de Game Bar OmniCharm

## Nekomata — Mapeo de mando personalizado por aplicación

Algunos juegos solo responden a teclado y ratón y no cuentan con soporte nativo para mando. Nekomata obra su magia en estas aplicaciones, remapeando cada entrada XInput (A/B/X/Y, LB/RB, LT/RT, LS/RS, cruceta, stick izquierdo, stick derecho) en acciones de teclado, ratón o cursor para que puedas jugarlos íntegramente con un mando.

Abre el editor desde el botón **"Personalizar el mapeo del mando para esta aplicación..."** del widget OmniCharm.

### Objetivos asignables

| Categoría | Objetivos |
|---|---|
| Letras | A–Z |
| Números | 0–9 |
| Teclas de función | F1–F24 |
| Edición / navegación / dirección | Insertar, Suprimir, Inicio, Fin, Retroceder página, Avanzar página, teclas de dirección, WASD |
| Teclas de control | Enter, Esc, Tabulador, Retroceso, Espacio, Impr Pant, Pausa, Bloq Mayús, Menú contextual (Apps) |
| Símbolos | ` - = [ ] \ ; ' , . / |
| Teclado numérico | Teclas numéricas 0–9, y . + - * / |
| Teclas multimedia | Volumen, controles de reproducción, teclas de navegador, teclas de inicio de aplicaciones |
| Combinación con modificadores | Ctrl / Shift / Alt / Win + tecla |
| Ratón | Botón izquierdo / derecho / central, rueda de desplazamiento, movimiento del cursor, desplazamiento |

## Widget OmniCharm

El widget de Game Bar OmniCharm te ofrece un acceso rápido dentro del juego a:

- La **Vista de tareas**, la **Biblioteca de Xbox** y la **interfaz superpuesta de Steam** como accesos directos de un solo toque.
- Activar o desactivar la **pulsación larga de ☰** para abrir la interfaz superpuesta en el juego de Steam.
- Activar o desactivar el **Modo de ratón por mando**, cambiar la distribución del controlador (**OmniNav / Clásica**) y ajustar la velocidad del cursor.
- Abrir el editor de mapeo de mando personalizado por aplicación de **Nekomata** para la aplicación en primer plano mediante el botón **"Personalizar el mapeo del mando para esta aplicación..."**.
